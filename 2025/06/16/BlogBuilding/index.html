博客搭建过程

关于我使用Hexo搭建个人博客的过程,参考流程：https://blog.fiveth.cc/p/bb32/

工具下载

包括Node.js以及git:

  • Hexo 是一个基于 Node.js 的博客框架，它使用 JavaScript 语言编写，并且依赖 Node.js 提供的运行环境来执行各种任务。Node.js 的安装会同时安装 npm（Node Package Manager），这是 Node.js 的包管理工具，用于安装、管理和配置 Hexo 及其相关的依赖包。
  • Git 是一个版本控制系统，可以帮助你管理博客的内容。在使用 Hexo 搭建博客的过程中，你可以使用 Git 来跟踪博客文章、配置文件、主题文件等的变更历史。这样，你可以方便地回溯到之前的版本，查看修改的记录，或者在出现问题时恢复到稳定的状态。
